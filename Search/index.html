Instructions

  • Click on the canvas to create a node
  • Shift click on the canvas to create a goal node
  • Shift click on a graph node to swap it with the current goal node
  • Click one node and then another node to create an edge between them
  • Double click a graph node to set it as the start node
  • Double click an edge and enter a value to assign a weight to it
  • Click a node or edge and press 'del' or 'backspace' to delete it
